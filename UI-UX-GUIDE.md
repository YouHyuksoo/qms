## Design System & User Interface Guidelines

**버전:** 1.0  
**작성일:** 2026-03-19  
**대상:** 자동차 부품 제조사 QMS 시스템

## 1. 디자인 철학 (Design Philosophy)

### 1.1 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **명확성 (Clarity)** | 품질 데이터는 정확하게, 오류는 명확하게 표시 |
| **효율성 (Efficiency)** | 현장 작업자가 빠르게 업무를 처리할 수 있는 UI |
| **신뢰성 (Trustworthiness)** | 데이터의 무결성을 시각적으로 전달 |
| **일관성 (Consistency)** | 모든 화면에서 동일한 패턴과 상호작용 |

### 1.2 사용자 페르소나

1. **현장 작업자 (Operator)**
   - 작업 환경: 생산라인, 검사실
   - 특성: 장갑 착용 가능, 빠른 데이터 입력 필요
   - 주요 화면: 검사 결과 입력, 부적합 보고

2. **품질 관리자 (Quality Manager)**
   - 작업 환경: 사무실
   - 특성: 다양한 데이터 분석, 리포트 확인
   - 주요 화면: 대시보드, 통계, 승인 업무

3. **경영진 (Executive)**
   - 작업 환경: 임원실
   - 특성: 핵심 지표 요약 확인, 의사결정 지원
   - 주요 화면: 경영 검토 대시보드, KPI

## 6. 인터랙션 가이드 (Interaction Guidelines)

### 6.1 폼 검증 (Form Validation)

#### 실시간 검증
- 입력 필드에서 포커스 아웃(blur) 시 즉시 검증
- 에러 메시지는 필드 하단에 빨간색 텍스트로 표시
- 성공 시 초록색 체크 아이콘 표시 (선택사항)

#### 제출 시 검증
- 에러가 있는 필드로 자동 스크롤
- 에러 필드에 포커스 및 흔들림 애니메이션

### 6.2 로딩 상태

```
버튼 로딩: 버튼 텍스트 대신 스피너 아이콘 회전
페이지 로딩: 스켈레톤 UI 또는 로딩 스피너
데이터 로딩: 테이블 행에 스켈레톤 효과
```

### 6.3 확인 대화상자

```
[삭제] 버튼 클릭 시:
┌─────────────────────────────┐
│  ⚠️ 삭제 확인               │
│                             │
│  이 데이터를 삭제하시겠습니까?│
│  삭제 후 복구할 수 없습니다.  │
│                             │
│        [취소]  [삭제]      │
└─────────────────────────────┘
```

### 6.4 알림 (Toast Notification)

```
위치: 화면 우측 상단

[성공] 녹색 배경, 체크 아이콘
```

## 15. 테마(Theme) UI 가이드

### 15.1 테마 모드

| 모드 | 설명 | 사용 상황 |
|------|------|----------|
| **Light** | 밝은 테마 | 기본, 사무실 환경 |
| **Dark** | 어두운 테마 | 야간, 어두운 공정 현장 |
| **System** | 시스템 설정 따라감 | 사용자 환경에 맞춤 |

### 15.2 테마별 색상 적용

```typescript
// ✅ CSS 변수 사용 (자동 테마 적용)
<div className="bg-background text-text-primary">
  <Card className="bg-surface border-border">
    <Button className="bg-primary-600 text-white">
      {t('common.save')}
    </Button>
  </Card>
</div>

// ❌ 하드코딩된 색상 금지
<div className="bg-white text-gray-900">  // 테마 변경 시 문제
```

### 15.3 테마 전환 UI

```
헤더 우측:
┌──────────────────────────────────────┐
│                      ☀️ 🌙 💻 👤    │

└──────────────────────────────────────┘
```

### 15.4 테마별 컴포넌트 예시

#### 버튼
```
Light Mode:
[확인] - primary-600 (#4F46E5), 흰색 텍스트

Dark Mode:
[확인] - primary-500 (#818CF8), 흰색 텍스트
```
